# 魔数+牛顿迭代法计算平方根

使用牛顿迭代法计算平方根。

$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$

我们计算平方根的公式：

$y = \sqrt{x} = x^{\frac 12}$

所以

${{y}^2} = x$

构建以y为自变量的函数方程为

$f(y) = {{y}^2} - x = 0$

$f'(y) = {2}{y}= 0$

将$f(y)$和$f'(y)$带入

$y - \frac{f(y)}{f'(y)} = y - \frac{{{y}^2} - x}{{2}{y}}$

​                 $= \frac{1}{2}(y + \frac{x}{y})$

i = 0x1fbd1df5 + (i >> 1);
